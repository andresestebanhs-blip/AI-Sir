# AI-Sir: Asistente Experto

Este bot ha sido entrenado con conocimientos avanzados en:
1.  **Programación**: Python, JavaScript, C++.
2.  **Ciberseguridad**: Hacking Ético, SQLi, XSS.
3.  **Escritura Creativa**: Fanfics, Estructuras Narrativas, Tropos.

## Instrucciones de Uso

1.  **Entrenar el Modelo**:
    Para actualizar el cerebro del bot con los nuevos datos expertos, ejecuta:
    ```bash
    python train.py
    ```
    *(Asegúrate de tener instalado PyTorch y NLTK)*

2.  **Iniciar el Chat**:
    Una vez entrenado (se creará `data.pth`), ejecuta:
    ```bash
    python chat.py
    ```
